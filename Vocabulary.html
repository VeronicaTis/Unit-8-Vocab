Unit 8 Vocabulary
Data Cells Contains content within columns or rows, By default, displays text as unformatted text and is aligned to teh left within the cell, Marked using the td element
td Used to define the data within a cell
table An HTML structure for creating rows and columns on a Web page
Web table A subset of the HTML tables on the Web contains relational data which can be useful for various applications.
rows a horizontal group of values within a table
columns a vertical group of values within a table
table cells is one grouping within a chart table used for storing information or data
header cell contain the information that is critical to understanding the raw data in a table. The -th- element needs to be used for header cells so that they are distinguishable from and can be associated with the correct data
th is the elemnet used to creat a header cell
CSS border property is used to add borders to any part of a web table
collapsed borders model Borders from adjacent elements are merged together to form a single border, border are joined to combine heir features, combining adjacent borders with different width, styles, or colors is complicated
Spanning cells Single cell that occupies more than one cell row and/or column, created by adding rowspan and/or colspan attributes to td or th elements, spanning starts in the cell with rowspan or colspan attributes and covers the cells in the right and below the initial cell
visual viewport the visual portion of a screen excluding on-screen keyboards, areas outside of a pinch-zoom area, or any other on-screen artifact that doesn't scale with the dimensions of a page
data attribute a characteristic of data that sets it apart from other data
block elements is any element that starts a new line, like a paragraph, and uses the full width of the page or container
two-column layout include a left column with navigation links, and a right column for body text
float elements places an element on the left or right side of its container, allowing text and inline elements to wrap around it
flexboxes Every HTML element that's a direct child of a flex container is an “item”. Flex items can be manipulated individually, but for the most part, it's up to the container to determine their layout
size the number of tables rows, and number of cells within rows
